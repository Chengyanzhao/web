Font some fonts

Source Code Pro

这是一款Adobe制作的字体，适用于系统使用。

Helvetica Neue

这是一款适合网页展示的字体

Icon some icons

Logoko

一款根据提供的要素定制图标的图标制作网站。

Easyicon

easyicon是一个图标搜索引擎工具，图标资源比较多，种类丰富，并且提供多种格式与尺寸下载。

Iconfont

阿里妈妈MUX倾力打造的矢量图标管理、交流平台。

Font Awesome

一套绝佳的图标字体库和CSS框架。

Iconmonstr

Iconmonstr专门提供黑白两色的矢量图标资源。

Find Icons

从这里你可以找到黑白/彩色、扁平/3D的图标。

Icon Archive

该网站的评级系统可以帮助你决定使用哪些图标比较好。

Iconfinder

该网站提供一个简单的过滤选项来帮助你寻找正确的图标。

The Noun Project

黑白纯色图标集。

Freepik

一款图标搜索引擎，另外提供整页的配套图标内容。

Fontello

这个网站可以让你创建自己的自由webfonts图标。选择你需要的图标符号，改变素大小，自定义文件名，下载。

Socialico

各种圆形风格的社交媒体图标。

Modern Pictograms

开放式字体下载。这里有很多富有吸引力的免费图标。图标都被设计得干净且可用性强，很适合于设计工作室或者字体网站设计。

Node Package some node package

Figlet

FIGlet是基于 ASCII 字符组成的字符画。支持多种字符画风格。并且提供了一个 测试工具 ，可以预览所有风格，并从中选择，非常方便实用！该项目已开源，可以去 GitHub上Fork一下。

ng2 Scroll To

该模块是一个ng2的指令，解决在ng2中a标签锚点无法使用的问题。只要在a标签上挂上'scrollTo'指令即可。本系统就是采用该模块实现锚点跳转功能的。另外这个模块也是学习ng2指令的一个好的教程。

Prism

Prism是一款多种风格，支持多种语言的代码高亮插件。并且提供了 在线试用工具。

网页模版 some page models
